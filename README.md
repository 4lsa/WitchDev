# WitchDev

macOS 窗口与浏览器标签页快速切换器，灵感来自 [Witch](https://manytricks.com/witch/)，完全免费开源。

## 核心优势

| 特性 | WitchDev | Witch (商业软件) |
|------|----------|-----------------|
| 价格 | **免费开源** | $14 起 |
| 拼音搜索 | ✅ **支持**（如 "gzt" 匹配 "工作台"） | ❌ 不支持 |
| Chrome 标签页 | ✅ **完整枚举所有标签** | ✅ 支持 |
| Ghostty 标签页 | ✅ **完整枚举所有标签** | ❌ 不支持 |
| 自定义快捷键 | ✅ 可修改 | ✅ 支持 |

## 快速开始

### 1. 安装

下载 `WitchDev.dmg`，将 `WitchDev.app` 拖到 `Applications` 文件夹。

> **注意**：由于使用 ad-hoc 签名，首次打开需允许：
> 1. 右键点击 WitchDev.app → 选择「打开」
> 2. 或在「系统设置 → 隐私与安全性 → 安全性」中点击「仍要打开」

### 2. 权限设置

首次使用需要授权两项权限：

**辅助功能**（用于监听全局快捷键 ⌥Tab）：
- 首次运行自动弹出授权对话框
- 前往「系统设置 → 隐私与安全性 → 辅助功能」勾选 WitchDev

**自动化**（用于获取 Chrome 标签页）：
- 首次按 ⌥Tab 呼出面板时，系统会提示"是否允许 WitchDev 控制 Google Chrome"
- 点击「确定」即可

### 3. 使用方法

| 操作 | 快捷键 |
|------|--------|
| 呼出切换器 | ⌥Tab (默认) |
| 向下导航 | ↓ / j / **Ctrl+N** |
| 向上导航 | ↑ / k / **Ctrl+P** |
| 进入搜索模式 | **f** |
| 按数字跳转 | ⌘1 ~ ⌘9 |
| 确认切换 | Enter / Space |
| 关闭面板 | Esc |

### 4. 拼音搜索（核心亮点）

按 **f** 进入搜索模式，输入拼音首字母即可快速定位：

- 输入 `gzt` → 匹配"工作台"
- 输入 `wd` → 匹配"文档"、"我的"等
- 支持中英文混合搜索

**Chrome/Ghostty 多标签示例**：

```
Google Chrome  □  工作台 - Team
              □  购买 Witch for mac - Google Search  
              □  消息
Ghostty        □  work | codeflicker
              □  Downloads
```

所有标签一目了然，无需逐个窗口切换。
